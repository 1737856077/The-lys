 刷新GIT：git fetch



git 放弃本地修改 强制更新[有效的]
git fetch --all
git reset --hard origin/master
git fetch 只是下载远程的库的内容，不做任何的合并 git reset 把HEAD指向刚刚下载的最新的版本
git pull命令的作用是:取回远程主机某个分支的更新

git reset --hard HEAD^   返回上一步操作
